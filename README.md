# Westongame React Boilerplate

## Dependencies:
- React
- React Router
- Redux
- PropTypes
- ClassNames
- Axios
- Uniqid
- Lodash

## Dev Dependencies:
- Webpack
- Webpack Dev Server
- ES2015+
- ESLint (Modified Airbnb)
- Jest + Enzyme
- CSSModules, Stylus, Autoprefixer

## How to start?
Before launching the project run  
`npm i`

Then, during development use  
`npm run dev`

## Other commands
- How to build production version  
`npm run build`

- How to lint  
`npm run lint`

- How to test  
`npm run test`
